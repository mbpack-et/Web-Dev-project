🎬 AnimeBase — Anime Movie & Series Database

Created by: Talaptan Alikhan, Mamedov Niyaz, Tynyshtyk Alis

📌 Project Description

AnimeBase is a web application designed for anime fans to explore, manage, and interact with a curated database of anime movies and series. The platform allows users to browse anime titles, view detailed information, leave ratings and comments, and manage their personal watchlists.

This project is built as a full-stack application using Angular (frontend) and Django + Django REST Framework (backend), with secure user authentication powered by JWT.
